# 栈（Stack）
栈是一种常用的数据结构,它是一种线性表，具有特定的操作规则。   
常用的操作包括压栈（Push）和出栈（Pop）。    

## 栈的定义如下：  

### 数据结构：  
栈是一种基于***后进先出***（`Last In First Out，LIFO`）原则的线性数据结构，可以用顺序存储结构（**数组**）或链式存储结构（**链表**）实现。
### 基本操作：   
__压栈__（Push）：将元素加入栈顶。如果栈已满，则称为栈溢出（Overflow）。  
__出栈__（Pop）：从栈顶移除元素。如果栈为空，则称为栈下溢（Underflow）。   
__获取栈顶元素（Top）__：获取栈顶元素的值，但不移除它。   
__判断栈空（isEmpty）__：判断栈是否为空。   
__清空栈（Clear）__：移除栈中的所有元素，使栈变为空栈。  
### 特点：  
栈只允许在一端***栈顶top***进行插入和删除操作，另一端***栈底***不可操作。  
后进入栈的元素先出栈，先进入栈的元素后出栈，因此栈的操作顺序为***后进先出***（`LIFO`）。  
### 应用：   
栈常用于实现函数调用的递归、表达式求值（如中缀表达式转后缀表达式）、括号匹配、浏览器的前进后退功能等。   
栈的抽象数据类型（ADT）定义可以用如下方式描述：   

数据：具有相同数据类型的元素的有限序列，元素称为栈的成员。   
### 操作：
`Push(x)：`将元素 x 压入栈顶。   
`Pop()：`弹出栈顶元素，并返回其值。   
`Top()：`返回栈顶元素的值，但不移除它。   
`isEmpty()：`判断栈是否为空，空则返回 `true`，否则返回 `false`。  
`Clear()：`清空栈，移除所有元素。   
栈的实现可以基于数组或链表，具体选择取决于应用场景和性能要求。数组实现的栈具有固定的容量，而链表实现的栈则具有动态的容量，可以根据需要进行扩展。
