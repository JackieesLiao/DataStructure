# 树

树（Tree）是一种非常重要的数据结构，它在计算机科学中有着广泛的应用。树由节点（node）组成，这些节点之间以边（edge）相连。  
树的一个特点是每两个节点之间只有唯一的路径。树常用于模拟层次结构，比如文件系统、组织架构等。      

## 树的基本概念和定义：  
节点（Node）：树的基本单元，树中的每个节点包含一个值，并且可能链接到其他节点。   
**根节点（Root Node）**：树顶部的节点，树的入口点。   
父节点（Parent Node）：一个节点的直接上级节点。   
子节点（Child Node）：一个节点的直接下级节点。    
叶子节点（Leaf Node）：没有子节点的节点称为叶子节点或终端节点。   
子树（Subtree）：树中的任意节点和它的后代构成的结构称为子树。  
深度（Depth）：节点的深度是指**从根节点到该节点的唯一路径上的边数**。     
高度（Height）：**节点的高度**是从**该节点到一个叶子节点**的最长路径上的边数。  
## 树的性质：
节点数量与边的关系：一棵有`n`个节点的树恰好有` n-1` 条边。  
路径唯一性：树中任意两个节点之间有且仅有一条路径。  
树的高度：树的高度等于从根节点到叶子节点的最长路径长度。   
节点的度数：**节点的度数**是指其*子节点的个数。*   
树的遍历：树的遍历有***深度优先遍历如前序、中序、后序遍历***和***广度优先遍历***两种方式。   

森林是m(m>=0)棵互不相交的树的集合。对树中每个节点而言，其子树的集合即为森林。

## 线性结构与树结构的区别  
### 线性结构   
1. 第一个数据元素无前驱  
2. 最后一个数据元素无后继  
3. 中间元素一个前驱和后继  
### 树结构  
1. 根节点无双亲，唯一   
2. 叶节点无孩子，可以有多个    
3. 中间节点，一个双亲多个孩子
